# Hastalık–Semptom İlişkisi Analizi

Python ile Birliktelik Kuralları Analizi kullanarak, ders kapsamında ekip çalışmasıyla hastalık–semptom ilişkilerini analiz eden bir veri madenciliği projesi geliştirdim.

## Kullanılan Yöntemler
- Apriori Algoritması
- Birliktelik Kuralları Analizi

## Değerlendirme Metrikleri
- Support (Destek)
- Confidence (Güven)
- Lift (Kaldıraç)

## Kullanılan Teknolojiler
- Python
- Pandas
- Mlxtend
- Jupyter Notebook

## Amaç
Belirli semptom kombinasyonlarının hastalıklarla olan ilişkilerini analiz ederek
anlamlı veri örüntüleri elde etmek.
